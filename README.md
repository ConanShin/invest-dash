# 📈 투자 자산 대시보드 (Investment Dashboard)

## 📝 프로젝트 소개 (Introduction)
이 프로젝트는 **국내주식, 미국주식, ETF** 등의 종목별 보유 좌수를 설정하고, 현재 시장 가격을 기준으로 **총 평가액**을 실시간으로 확인할 수 있는 대시보드입니다. 통합된 자산 현황을 한눈에 파악하여 효율적인 투자 관리를 돕는 것을 목표로 합니다.

This project is a dashboard that allows users to configure their holdings for **Domestic Stocks, US Stocks, and ETFs**, and view the **total valuation** based on current market prices.

## ✨ 주요 기능 (Key Features)
- **자산 보유량 설정**: 종목별 보유 수량(좌수) 입력 및 관리
- **실시간 가치 평가**: 현재 증시 데이터를 기반으로 평가금액 계산
- **포트폴리오 현황**: 자산군(국내/미국/ETF)별 비중 및 현황 시각화
- **통합 대시보드**: 총 자산 규모를 직관적으로 확인

## 🛠️ 기술 스택 (Tech Stack)
- **Framework**: Flutter (Dart)
- **State Management**: Riverpod
- **Local Database**: Drift (SQLite)
- **Architecture**: MVVM / Clean Architecture
